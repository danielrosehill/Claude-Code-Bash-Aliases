# Create Claude Commands and Agents Directories

Creates both `.claude/commands` and `.claude/agents` directories in the current working directory for project-level Claude Code configuration.

```bash
alias commandsandagents='mkdir -p .claude/commands .claude/agents && echo "Created .claude/commands and .claude/agents"'
```

## Usage

```bash
commandsandagents
```

This will create both directories needed for Claude Code slash commands and agents at the project level.
